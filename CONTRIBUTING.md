# Contributing to ESP8266 Wi-Fi Deauther

Thank you for considering contributing to this project! Here's how you can help.

## 🐛 Reporting Bugs

1. Check the [Issues](../../issues) page to see if the bug has already been reported
2. If not, open a new issue with:
   - A clear title and description
   - Steps to reproduce the bug
   - Expected vs actual behavior
   - Your hardware (board model, display type, etc.)
   - Arduino IDE version and ESP8266 core version

## 💡 Suggesting Features

Open an issue with the `enhancement` label and describe:
- What the feature does
- Why it's useful
- How it could be implemented (optional)

## 🔧 Submitting Code

1. **Fork** the repository
2. **Create a branch** for your feature: `git checkout -b feature/my-feature`
3. **Make your changes** — follow the existing code style
4. **Test** on real hardware if possible
5. **Commit** with clear messages: `git commit -m "Add: description of change"`
6. **Push** to your fork: `git push origin feature/my-feature`
7. **Open a Pull Request** against the `main` branch

## 📏 Code Style

- Use consistent indentation (4 spaces)
- Keep functions focused and well-named
- Add comments for complex logic
- Follow existing patterns in the codebase

## ⚖️ Code of Conduct

- Be respectful and constructive
- **This tool is for educational and authorized security testing only** — do not contribute features intended for malicious use

Thank you for helping improve this project! 🎉
